# Client - FrontEnd

A pasta "client" engloba o lado do cliente da aplicação, referindo-se especificamente ao frontend. O lado do cliente(ou client-side) é a parte do sistema que é executada diretamente no navegador do cliente.

# Server - BackEnd/Api

O diretório "server" representa o backend da aplicação, geralmente mencionam como server-side. A pasta engloba os "sistemas internos" da aplicação e, principalmente a API.
